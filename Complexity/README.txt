The project was carried out in Python notebooks.The main file is Complexity_Project.ipynb. There,
you will find in implementation of the Oslo Model along with all the functions used to collected
data from the system. The data is saved either in a CSV file or a series of text files depending
on the application. These are processed in the Data_Analysis_Height.ipynb or 
Data_Analysis_Avalanche_Size_Probability.ipynb for the relevant parts of the project.
